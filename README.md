## Notes

### [`<audio>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/audio)

Reset time to start point:
```javascript
$audio.currentTime = 0
```

Seek back to start point when reaching the end:
```html
<audio loop></audio>
```
